Tách Dữ Liệu Offline (Cold Archive)

LƯU Ý QUAN TRỌNG: Quá trình này sẽ trích xuất và XÓA VĨNH VIỄN dữ liệu gốc từ năm đã chọn khỏi Server!

* Bỏ chọn (Cập nhật Mềm): Các máy đã có file Offline cũ vẫn tiếp tục dùng. Chỉ máy tính mới hoặc mất dữ liệu mới phải nạp file này.

YÊU CẦU CẬP NHẬT FILE DỮ LIỆU

Hệ thống phát hiện có bản cập nhật Dữ liệu Offline (Cold Archive) từ Server. Để đảm bảo số liệu thống kê đầy đủ, bạn cần nạp file gốc mới nhất trước khi tính toán.

Hướng dẫn:
1. Tải file Dữ liệu Offline mới nhất tại đây: Tải File Offline
2. Lưu file và giải nén (nếu là file nén).
3. Bấm nút bên dưới để chọn file vừa tải về (giải nén trước nếu là file nén) để nạp dữ liệu vào hệ thống.
